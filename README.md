## Introduction

<p align="justify">
In today’s digital world, authentication system plays a very important role in ensuring the security of software applications and information systems. Almost every application, whether it is a website, desktop application, or mobile app, requires a sign in (login) mechanism to verify the identity of users before granting access.</p>

<p align="justify">
The project titled “Sign In Form” focuses on creating a simple, secure, and user-friendly sign-in system using Python programming language. The project provides a graphical user interface (GUI) that allows users to enter their username and password for authentication.</p>

<p align="justify">
The sign-in form validates user input and checks the credentials to ensure that only authorized users can access the system. The interface is designed using Python GUI libraries such as Tkinter and CustomTkinter, which makes the application interactive and visually appealing.</p>

<p align="justify">
The main aim of this project is to understand the basic concepts of GUI development, user authentication, and input validation using Python. This project also helps in gaining practical experience in designing real-world applications and improves programming skills.</p>

---

## Working of the Sign in form System

<p align="justify">
  
**Username or Email**

Username or Email, this is an entry widget that accepts user input, such as Ajay Kumar or ajay1234@gmail.com. It uses the placeholder text attribute for modern styling.

  
**Password** 

Password is an entry widget that accepts input as a password, for example: ajay12345. It also provides password hiding feature to ensure the security of user's password.</p>

**Forgot Password**

Forgot Password is a link that allows users to create a new password if they have forgotten their previous password.

**Sign in Button**

The Sign In button is a send button. When the user fills in all the fields and clicks the button, a success message or notification is generated. If the user clicks the Sign In button without filling in all the fields, an error message is generated.

**Sign in link**

The Sign In link allows the user to create a new account if they don't already have one. 

**Social Media Links**

Social media links allow the user to sign in through the provided social media account links such as Google, Facebook and Instagram.

**User Friendly Image**

To make the graphical user interface more attractive, an image has been attached with the sign in form system which makes it modern and attractive to look at.
</p>

## Technologies Used

### Programming Language Used:

**Python**

Python is high-level, interpreted programming language known for its simplicity and readability. It is widely used for application development, automation, and GUI-based software development. In this project, Python is used to develop Sign In Form and implement authentication logic.

### Libraries and Modules Used:

**CustomTkinter**

CustomTkinter is a modern and customizable Python UI library based on the standard Tkinter library. It provides a set of updated widgets that allow developers to create visually appealing, professional-looking, and consistent graphical user interfaces (GUIs) across different desktop platforms (Windows, macOS, Linux). It is used in this project to design a visually attractive and responsive Sing in interface. 

**Install Instructions:**
 
```bash
pip install customtkinter
```

<br>

**CTkImage (from CustomTkinter)**

CTkImage in Python’s CustomTkinter library, is a utility class designed to handle and display images in a way that allows for proper scaling on High- DPI displays and support different images for light and dark appearance modes. It helps in adding icons and background images to enhance the user interface. 

<br>

**PIL (Python Imaging Library)**

The PIL is an open-source library that adds image processing capabilities to the python interpreter. The original PIL project is no longer actively developed, and its modern actively maintained successor is know as Pillow. When working with image processing in python today, you need to install pillow package.

**Install Instructions:**

```bash
pip install pillow
```

<br>

**Tkinter Messagebox**

In Python, a messagebox is a modal dialog box used in graphical user interface (GUI) application to display short messages, notifications, warnings, error, or to ask the user a simple question for confirmation. Its most commonly implemented using the Tkinter’s messagebox module, which is part of python’s standard tkinter library for GUI development.

**Install Instructions:**

```bash
pip install  messagebox
```

<br>

**Webbrowser Module**

Webbrowser module is used to open external links such as social media pages or websites directly from the applications.
It works across different operating systems (Windows, macOS, Linux) by using the default browser configured on each system. 

**Install Instructions:**

```bash
pip install web-browser
```
